# Clue Game - Local Test Environment

Fast local testing setup for the Clue Game web application.

## Setup

1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

2. Run the game:
   ```bash
   python main.py
   ```

3. Open in browser:
   http://127.0.0.1:5000

## Features

- Debug mode enabled for faster development
- Auto-reload on file changes
- Local host binding for security
- Port 5000 for easy access

## Directory Structure

- `main.py` - Entry point for local testing
- `web/` - Flask web application
- `templates/` - HTML templates
- `src/` - Game logic and engine
